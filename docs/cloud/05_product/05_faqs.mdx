# FAQs

<details>
  <summary>What is Glasskube Cloud?</summary>
  <p>Glasskube Cloud is an open-core commercial software distribution platform designed for Independent Software Vendors (ISVs) to manage and deploy software to customer-controlled environments.</p>
</details>

<details>
  <summary>How do Glasskube Cloud agents work?</summary>
  <p>Glasskube Cloud agents are lightweight components that are installed in the target deployment environment and can be delivered by Glasskube Cloud or embedded directly into ISV Helm charts or Docker Compose files. They handle lifecycle tasks such as installation, updates, and rollbacks while providing essential metrics and logs.</p>
</details>

<details>
  <summary>Can I customize the Glasskube Cloud platform for my brand?</summary>
  <p>Yes, Glasskube Cloud can be white-labeled with your branding, providing a consistent experience for your customers and making the platform appear as an extension of your product suite.</p>
</details>
